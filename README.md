Description of the project: Establishing a copy of the AirBnb website with a certain number of it's feautures.
Description of the command interpreter: A program that allows users to interact with a system or application by entering commands in a terminal or console.
The interpreter parses the user input and executes the corresponding actions. To start the command interpreter run the following command
$ python interpreter.py
You use it with the following syntax 
command_name arg1 arg2 ...
Examples; $ mkdir my_folder
$ cd my_folder
$ rm file.txt
$ ls
